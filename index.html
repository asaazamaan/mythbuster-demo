MythBuster

Automated Fact-Checking for Arabic Videos About Diabetes

Demo

The core demo is a short recorded walkthrough showing an end-to-end analysis of one Arabic diabetes-related video. (An embedded video will appear here once uploaded.)

Watch demo GitHub repository

What the demo shows

In the recorded demo, you will see:

  • Submitting a short Arabic diabetes-related video URL
  • Automatic transcription of the spoken Arabic content
  • Extraction of check-worthy medical claims related to diabetes
  • Generation of fact-checking verdicts (TRUE / FALSE / PARTIALLY TRUE / INSUFFICIENT INFO)
  • Evidence-grounded Arabic medical explanations that justify the verdicts by synthesizing retrieved medical knowledge and explaining the reasoning behind each decision
  • Retrieval of supporting medical evidence with clickable links to trusted sources, including evidence retrieved via web search and retrieval-augmented generation (RAG)

Tech stack

  • Backend: FastAPI (API orchestration and request handling)
  • Frontend: React (interactive results visualization)
  • Database: PostgreSQL (analysis results storage and caching)
  • Retrieval: RAG + trusted web search (evidence grounding)
  • DevOps: Docker (containerized full-stack setup)
